Hard Skills

Hard Skills are skills related to working with technology directly.
The best way to improve these is to watch tutorials, practice, and gain experience!

Tutorials online (youtube and others) are really helpful! The internet is an INCREDIBLY powerful resource for computer scientists!

Some example hard skills (there are more) are:

    The primary skill for any developer, what languages you know directly influence your job abilities. What's most important is to know one very well, that way you can apply the skills you learn to all the others! Some popular languages include: Python, C++, Java, and Javascript.

    A subset of programming languages, frameworks and libraries simplify the jobs of developers. Knowing popular ones may open doors for job opportunities, or give you insights on ways to simplify projects. These exist for any language, so find an interest you have, and look into popular frameworks!

    There are many ways to solve a problem in Computer Science. Finding the best way possible and being able to articulate that is an incredibly important skill, especially if you work in a small team.

    Almost all software needs to be properly tested before end users access it. Finding unintended behavior, workarounds, and exploits can ensure safer, better code is given to users. Knowing how to find weaknesses in code, exploit, and demonstrate them will be great for any QA or cyber career.

    All code has to run somewhere. Knowing the key components for servers and hardware is an important field in Computer Science. Most developers don't like messing with infrastructure things, so having skills might set you apart!

    Huge amounts of data are stored online. Knowing how different databases work will be a great skill in the Information Age. Being able to query the data to produce meaningful results will set you on a great path for database administration or data science.

    Some incredibly important functions are now run through the internet. Things like banking, payments, and heaps of personal information are stored online. Knowing how to secure this data keeps your customers safe and prevents lawsuits. Win-win!
