Soft Skills

Soft Skills are often related to the work you'll do as a computer scientist, but aren't directly related to technology usage. Employers still value these!
Just like any other skills, the best way to improve is to practice!

Some example soft skills (there are more) are:

  • Problem Solving
    • This one is particularly important for a computer scientist!
  • Detail-oriented
  • Fast learner
  • Communication
  • Presentations
  • Teamwork
  • Time management
  • Critical thinking
  • Flexibility
